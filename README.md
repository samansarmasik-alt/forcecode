# 🚀 ForceCode

> **A lightweight, open-source AI Coding Agent with support for 22+ AI providers.**

ForceCode is a terminal-based AI coding assistant designed to help you build, edit, and manage software projects using the latest language models. It supports multiple AI providers, custom OpenAI-compatible APIs, intelligent agents, project memory, web search, and much more.

---

## ✨ Features

* 🤖 Support for **22+ AI providers**
* 🔌 Custom OpenAI-compatible provider support
* 💬 Interactive terminal interface
* 🧠 Multi-agent workflow
* 🌐 Built-in web search
* 📦 Automatic model & pricing discovery
* 🛠️ Project memory and persistent instructions
* 📄 Project initialization (`init`)
* ⚡ Build & Plan modes
* 🔒 Safe command execution modes
* 🎯 Lightweight and fast
* 🐍 Built with Python
* ❤️ Fully Open Source

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/samansarmasik-alt/forcecode.git
cd forcecode
```

Install ForceCode globally:

```bash
install_force.bat
```

After installation you can simply run:

```bash
force
```

Or launch it directly without installation:

```bash
start_force.bat
```

## ⚙️ Key Commands

| Command  | Description                 |
| -------- | --------------------------- |
| `force`  | Start ForceCode             |
| `init`   | Initialize project memory   |
| `memory` | Save project knowledge      |
| `forget` | Remove saved memory         |
| `prompt` | Edit system instructions    |
| `logs`   | View operation logs         |
| `status` | Show current project status |

## 🧩 Build Modes

ForceCode includes multiple execution modes:

* **Plan Mode** – Think before coding
* **Build Mode** – Focus on implementation
* **Multi-Agent Mode** – Use multiple AI agents
* **Safe Execution** – Ask before running sensitive commands
* **Automatic Execution** – Fully autonomous workflow

## 💡 Why ForceCode?

Unlike many AI coding assistants, ForceCode gives you the freedom to choose your preferred AI provider without being locked into a single ecosystem.

* Use your own API keys.
* Switch providers instantly.
* Compare models easily.
* Stay lightweight.
* Stay open source.
